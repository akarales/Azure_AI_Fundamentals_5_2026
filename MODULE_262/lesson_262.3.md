## Section 1: Speech Recognition and Synthesis

### Speech-to-Text

Speech-to-text (speech recognition) converts spoken language into written text. It enables hands-free interaction with applications and is used in transcription, voice commands, and accessibility tools.

### Text-to-Speech

Text-to-speech (speech synthesis) converts written text into spoken audio. It powers virtual assistants, reading aids, and interactive voice response (IVR) systems.

### Azure AI Speech Service

Azure AI Speech provides both speech-to-text and text-to-speech capabilities:

- **Real-time transcription:** Convert live speech to text.
- **Batch transcription:** Process pre-recorded audio files.
- **Custom Speech:** Train models with your own audio data for improved accuracy.
- **Neural voices:** Generate natural-sounding speech with AI-generated voices.
- **Speaker recognition:** Identify and verify speakers based on their voice.

**Reference:** [Microsoft Learn - Get Started with Speech in Azure](https://learn.microsoft.com/en-us/training/modules/get-started-speech-azure/)

## Section 2: Language Translation

Language translation is the process of converting text or speech from one language to another. AI-powered translation systems use neural machine translation for high-quality results.

### Azure AI Translator Service

Azure AI Translator provides real-time text translation across 100+ languages:

- **Text Translation:** Translate text between languages via API.
- **Document Translation:** Translate entire documents while preserving formatting.
- **Custom Translator:** Train custom translation models with domain-specific terminology.
- **Transliteration:** Convert text from one script to another (e.g., Arabic to Latin).

**Reference:** [Microsoft Learn - Translate Text with Translator Service](https://learn.microsoft.com/en-us/training/modules/translate-text-with-translator-service/)

### Real-Time Translation

Real-time translation enables live conversation translation across languages, supporting multilingual meetings, customer support, and global collaboration.
