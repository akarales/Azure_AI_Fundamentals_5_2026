## Skills Measured (as of April 15, 2026)

### Domain 1: Identify AI Concepts and Capabilities (40-45%)

**Describe principles of responsible AI:**
- Fairness, reliability, safety, privacy, security, inclusiveness, transparency, accountability

**Identify AI model components and configurations:**
- Describe how generative AI models work
- Identify an appropriate AI model based on capabilities
- Identify appropriate model deployment options and configuration parameters

**Identify AI workloads:**
- Scenarios for common AI workloads (generative AI, agentic AI, text analysis, speech, vision, information extraction)
- Common text analysis techniques (keyword extraction, entity detection, sentiment analysis, summarization)
- Speech recognition and synthesis features
- Computer vision and image-generation model features
- Techniques to extract information from text, images, audio, and videos

### Domain 2: Implement AI Solutions by Using Microsoft Foundry (55-60%)

**Implement generative AI apps and agents by using Foundry:**
- Create effective system and user prompts for generative AI models
- Deploy a model and interact with it in the Foundry portal
- Create a lightweight chat client application by using the Foundry SDK
- Create and test a single-agent solution in the Foundry portal
- Create a lightweight client application for an agent

**Implement AI solutions for text and speech by using Foundry:**
- Build a lightweight application that includes text analysis
- Respond to spoken prompts by using a deployed multimodal model
- Build a lightweight application by using Azure Speech in Foundry Tools

**Implement AI solutions with computer vision and image-generation by using Foundry:**
- Interpret visual input in prompts by using a deployed multimodal model
- Create new visual outputs by using generative models
- Build a lightweight application that includes vision capabilities

**Implement AI solutions for information extraction by using Foundry:**
- Extract information from documents and forms by using Azure Content Understanding in Foundry Tools
- Extract information from images by using Content Understanding
- Extract information from audio and video by using Content Understanding
- Build a lightweight application with information extraction capabilities

## What's Different About AI-901 (Compared to AI-900)

### Removed from AI-900 (not on AI-901):
- Azure Machine Learning workspace and AutoML (hands-on)
- Custom Vision service
- Azure Cognitive Search (traditional index-based)
- Language Studio (CLU, Question Answering as standalone services)
- Cloud Shell scripting labs

### Added for AI-901 (not on AI-900):
- Microsoft Foundry portal navigation and project creation
- Model deployment and configuration in Foundry
- Foundry SDK (Python) for chat client applications
- Single-agent creation and testing in Foundry
- Agent client applications
- Content Understanding (replaces Document Intelligence and OCR)
- Foundry IQ (replaces traditional search/QA)
- Image generation models
- Video generation (Sora)
- Multimodal model prompting (text + image input)

### Still on both exams:
- Responsible AI principles
- AI workload identification (vision, NLP, speech, extraction)
- Text analysis techniques
- Speech recognition and synthesis
- Computer vision concepts
- Generative AI fundamentals

## Quick Start Checklist for AI-901

If you plan to take AI-901 after passing AI-900 (or instead of AI-900):

- [ ] Review the [AI-901 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-901)
- [ ] Complete Learning Path: [Get started with AI applications and agents on Azure](https://learn.microsoft.com/en-us/training/paths/get-started-ai-apps-agents/)
- [ ] Complete all 6 hands-on labs from `mslearn-ai-fundamentals` (listed above)
- [ ] Practice Python SDK basics (Foundry SDK for chat completions and agents)
- [ ] Explore the Microsoft Foundry portal at [ai.azure.com](https://ai.azure.com)
- [ ] Review responsible AI principles (same as AI-900)
- [ ] Try the exam sandbox when practice assessments become available
