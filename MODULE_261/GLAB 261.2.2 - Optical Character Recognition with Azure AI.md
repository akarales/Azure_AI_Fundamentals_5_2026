# **Introduction**

In this lab, you will explore the Optical Character Recognition (OCR) capabilities of Azure AI Vision. You will extract text from images and documents using the Azure AI Read API.

# **Objectives**

* Explore OCR capabilities in Azure AI Vision.
* Extract printed and handwritten text from images.
* Understand how the Read API processes documents.
* Apply OCR to real-world document processing scenarios.
