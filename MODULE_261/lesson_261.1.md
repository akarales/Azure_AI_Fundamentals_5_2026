## Section 1: What is Computer Vision?

Computer Vision is a field of AI that enables machines to interpret, understand, and make decisions based on visual data such as images and videos. It allows computers to "see" and extract meaningful information from the visual world.

### Key Computer Vision Tasks

- **Image Classification:** Categorizing an image into predefined classes (e.g., identifying whether an image contains a cat or a dog).
- **Object Detection:** Identifying and locating objects within an image with bounding boxes.
- **Optical Character Recognition (OCR):** Extracting text from images and documents.
- **Facial Detection and Analysis:** Detecting human faces in images and analyzing attributes such as age, emotion, and identity.
- **Image Segmentation:** Dividing an image into distinct regions based on content.

---

## Section 2: Image Classification

Image classification assigns a label to an entire image from a set of predefined categories. This is one of the most fundamental computer vision tasks.

### Applications
- Medical imaging (classifying X-rays or MRI scans)
- Quality control in manufacturing
- Wildlife monitoring

---

## Section 3: Object Detection

Object detection not only classifies objects within an image but also provides their precise locations using bounding boxes.

### Applications
- Autonomous vehicles (detecting pedestrians, signs, other vehicles)
- Retail analytics (tracking products on shelves)
- Security and surveillance

---

## Section 4: Optical Character Recognition (OCR)

OCR enables the extraction of printed or handwritten text from images, scanned documents, and PDFs.

### Applications
- Digitizing paper documents
- Processing receipts and invoices
- Extracting text from photos of signs or menus

---

## Section 5: Facial Detection and Analysis

Facial detection identifies the presence and location of human faces in an image. Facial analysis can determine attributes such as age, emotion, and head pose.

### Applications
- Photo organization
- Security and access control
- Customer experience analysis

---

## Section 6: Azure AI Vision Services

Microsoft Azure provides powerful computer vision capabilities through several services:

- **Azure AI Vision:** Image analysis, OCR, and spatial analysis.
- **Azure AI Face:** Face detection, verification, and grouping.
- **Azure AI Custom Vision:** Build and train custom image classification and object detection models.
