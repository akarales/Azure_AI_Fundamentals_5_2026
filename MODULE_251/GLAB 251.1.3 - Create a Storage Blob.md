# **Introduction**

In this lab, you will explore Azure Storage by creating a container, uploading a file (blob), and accessing it through its public URL. This exercise helps build foundational knowledge of Azure Blob Storage and how it supports scalable, secure object storage in the cloud.

# **Objectives**

* **Create** and **configure** a storage container in Azure.
* **Upload** and **manage** blob data using the Azure portal.
* **Validate** public access to blobs using URL-based access.

# **Instructions**

You are working as a cloud administrator tasked with organizing files in the cloud using Azure Blob Storage. Your goal is to create a storage container, upload a file to it, and test access using the public URL.
