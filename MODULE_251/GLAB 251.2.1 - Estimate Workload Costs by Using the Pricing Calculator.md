# **Introduction**

In this lab, you will explore Azure's cost planning tools by using the Azure Pricing Calculator to estimate monthly expenses for virtual machines and storage resources. This exercise supports cloud budget forecasting and helps build essential cost-management awareness.

# **Objectives**

* **Navigate** and **configure** estimates using the Azure Pricing Calculator.
* **Calculate** workload costs based on VM and storage configurations.
* **Interpret** how pricing varies by region, redundancy, and usage patterns.

# **Instructions**

You are working as a cloud consultant tasked with generating a preliminary cost estimate for deploying infrastructure in Azure. You will use the Azure Pricing Calculator to estimate the monthly costs of a virtual machine and a storage account, and then capture the final pricing output for reporting.

- **Microsoft Learn Lab URL:** [https://learn.microsoft.com/en-us/training/modules/analyze-costs-create-budgets-azure-cost-management/](https://learn.microsoft.com/en-us/training/modules/analyze-costs-create-budgets-azure-cost-management/)

You will:

* Add a virtual machine configuration to the pricing estimate
* Add a storage account to the same estimate
* Capture the final total monthly cost after customizing the configuration
