# **Introduction**

This lab is designed to provide a guided experience in deploying a virtual machine (VM) using **Azure Lab Services** and the **Azure Portal**. Learners will deploy resources, examine associated components, and apply real-world concepts in a controlled sandbox environment.

# **Objectives**

* **Identify** the essential components and roles involved in Azure Lab deployment.
* **Describe** the architecture of Azure Lab Services and the Azure VM portal experience.
* **Execute** a full deployment of a lab or virtual machine using provided documentation.
* **Inspect** Azure Resource Groups to verify linked infrastructure.
* **Evaluate** the configuration and implications of network rules and resource settings.
* **Create** a lab environment tailored to specific educational or testing goals.
