# **Introduction**

In this lab, you will explore how to use the Azure Total Cost of Ownership (TCO) Calculator to estimate and compare the costs of running on-premises workloads versus moving them to Azure. This tool helps you assess potential cost savings over a multi-year period.

# **Objectives**

* Input on-premises workload details into the Azure TCO Calculator.
* Analyze estimated cost differences between on-premises infrastructure and Azure services.
* Interpret summary reports to support cloud cost justification.
