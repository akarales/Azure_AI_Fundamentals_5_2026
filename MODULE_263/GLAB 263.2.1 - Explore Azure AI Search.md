# **Introduction**

In this lab, you will explore Azure AI Search and its knowledge mining capabilities. You will create a search index, apply AI enrichment, and query data using full-text search.

# **Objectives**

* Create an Azure AI Search resource.
* Build a search index with AI enrichment.
* Query the index and review results.
* Understand how knowledge mining extracts insights from unstructured data.
