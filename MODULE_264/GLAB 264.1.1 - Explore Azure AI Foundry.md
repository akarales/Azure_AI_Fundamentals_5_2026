# **Introduction**

In this lab, you will explore Azure AI Foundry (formerly Azure AI Studio), the unified platform for building generative AI applications. You will navigate the interface, explore available models, and use the chat playground to interact with a language model.

# **Objectives**

* Navigate the Azure AI Foundry portal.
* Explore the model catalog and deployment options.
* Use the chat playground to interact with a language model.
* Understand how to configure system messages and parameters.
