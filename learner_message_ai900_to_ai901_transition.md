## Current Status of the AI-900 Exam

Here is what you need to know:

**The AI-900 exam (Azure AI Fundamentals) is retiring on June 30, 2026.**

Microsoft announced that the AI-900 exam will be replaced by a new exam called **AI-901** (also called "Azure AI Fundamentals" but with completely different content). The AI-901 is currently in **beta** and became available in April 2026.

### What this means for you:

1. **Your certification is still valid.** If you pass AI-900 before June 30, 2026, you earn the "Microsoft Certified: Azure AI Fundamentals" certification. This certification does NOT expire. It is yours permanently.

2. **All our course labs still work.** I verified every single lab URL today. They are all accessible and will remain so through June 2026.

3. **The content we studied is still relevant.** AI concepts, machine learning, computer vision, NLP, document intelligence, and generative AI are all still on the AI-901 exam. The difference is that AI-901 focuses more on Microsoft Foundry (the new platform) and less on individual services.

## Should You Take AI-900 or Wait for AI-901?

This is a personal decision, but here is my honest assessment:

### Take AI-900 NOW if:
- You are confident in the material we covered
- You want to get certified before June 30, 2026
- You want a proven, stable exam with well-known question patterns
- You want to use the practice exams and qualification exams we prepared (they are calibrated for AI-900)

### Consider waiting for AI-901 if:
- You need more time to study
- You are comfortable with Python code (AI-901 requires familiarity with Python syntax)
- You want to demonstrate skills with the latest Microsoft Foundry platform
- You are okay taking a beta exam (beta exams can have some rough edges, and results may take longer)

### Key differences between AI-900 and AI-901:

| | AI-900 (retiring June 30) | AI-901 (beta, available now) |
|--|---|---|
| **Focus** | AI concepts + individual Azure AI services | AI concepts + Microsoft Foundry platform |
| **Coding** | No coding required | Python familiarity required |
| **Exam Weight** | AI Concepts (15-20%), ML (20-25%), Vision (15-20%), NLP (15-20%), GenAI (20-25%) | AI Concepts (40-45%), Foundry Implementation (55-60%) |
| **Lab Style** | Portal-based, point-and-click | SDK-based, code-along |
| **Certification Earned** | Azure AI Fundamentals | Azure AI Fundamentals (same cert) |
| **Practice Assessment** | Available | Not yet available (usually 8 weeks after GA) |

### My recommendation:
If you have been keeping up with the material, **take AI-900 now**. You have the knowledge, you have the practice exams, and you have a clear path to certification. There is no advantage to waiting, the certification you earn is identical regardless of which exam you pass.

If you feel you need more time, AI-901 will still be available after June 30. You will earn the same certification. But you will need to study additional content around Microsoft Foundry and Python SDK usage.
